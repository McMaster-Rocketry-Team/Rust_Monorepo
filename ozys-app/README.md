# OZYS App

A desktop app for connecting to OZYS PCBs and visualizing real-time strain gauge data.

## Getting Started

### Prerequisites
Make sure you have installed nvm, yarn and Rust.

### Installing

```
cd ozys-app

# Use node version 22
nvm use 22

# enable corepack
corepack enable

# Install dependencies
yarn install
```

### Executing program

To run the app:
```
yarn tauri dev
```
